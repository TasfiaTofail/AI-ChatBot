 🤖 MiniLang Chatbot (Flask)

A simple rule-based chatbot built using **Flask** that responds to user messages through a web interface and a REST API.

## 🚀 Features

- 🌐 Web-based chat interface (HTML frontend)
- ⚡ Flask backend API
- 💬 Rule-based chatbot responses
- 🧠 Simple keyword detection (hello, how are you, joke, etc.)
- 🔄 JSON-based communication between frontend and backend

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌶️
- HTML/CSS (Frontend)
- JavaScript (for API calls)
